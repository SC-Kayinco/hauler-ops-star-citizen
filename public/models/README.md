# Ship models

Drop `.glb` / `.gltf` ship models here, then set a ship's **Model file** field
(in Ship → Edit Ship) to `/models/your-file.glb`.

The app applies a holographic material automatically. If no model is set, a
procedural placeholder hull is shown instead.

Tip: generate a model from a screenshot with Tripo3D or Scenario, export `.glb`,
and place it in this folder.
